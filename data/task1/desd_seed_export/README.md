# AI Task 1 DESD Export

This export contains fake/seed/demo marketplace data for the Advanced AI Task 1 quick-reorder proof of concept.

It is not real customer behaviour and must not be used to claim production recommender performance.

## Privacy

Customer names, emails, phone numbers, full addresses, and delivery addresses are not exported.
Producer contact email, phone, full address, bank/payment data, and private profile fields are not exported.

For future real data exports, replace readable demo IDs with salted hashes.
